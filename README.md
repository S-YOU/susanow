
# Susanow

A high performance network-stack on DPDK.
Please see https://susanow.github.io


## Usae

```
$ git cloen http://github.com/susanow/susanow.git
$ cd susanow/src
$ make rr
$ make run
```

## Required Software

 - make
 - g++
 - dpdk-16.07
